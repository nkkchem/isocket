Overview

The Atlas of Coiled Coils is a fully automated, updated overview of extant coiled coils.

Coiled coils are categorised using graph theory.
Each helix in the coiled coil is represented by a node.
Helices that interact via knob-into-hole interactions are joined by an edge.
This gives the graph of the coiled coil: those with the same graphs belong to the same category.


Details

The classification is implemented in Python.
For identifying coiled coils, we use the ISAMBARD (Intelligent System for Analysis, Model Building And Rational Design) Python package developed in the Woolfson group.

The SOCKET cutoff (scut):
The maximum distance between the centre of mass of a knob residue and those for each of the hole residues

The knob cutoff (kcut):
There must be more than kcut knob-into-hole interactions between each pair of helices defined to be interacting.
